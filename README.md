# guide-mk-cns

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库本身不针对任何特定的域名或网站，而是作为一个通用的页面集合，用于存放各类独立的网页资源。

## 目录说明

```
/
├── pages/          # 存放独立 HTML 页面
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
├── docs/           # 相关文档或说明文件
└── README.md       # 本文件
```

- **pages/**：主要页面存放目录，每个 HTML 文件为一个独立页面。
- **assets/**：页面所需的样式、脚本及媒体文件，按需组织。
- **docs/**：用于存放项目的补充说明或技术文档。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，不依赖外部服务或特定域名。每个页面可单独通过文件路径或 GitHub Pages 等静态托管服务访问。

页面内容可能包含文字、图片、样式及交互脚本，但所有资源均已打包或内联，以确保页面在离线或本地环境下也能正常展示。

## 使用方式

你可以通过以下方式使用本仓库中的页面：

1. **直接浏览**：克隆仓库后，在浏览器中打开 `pages/` 目录下的 HTML 文件。
2. **GitHub Pages 托管**：启用仓库的 GitHub Pages 功能，页面将通过 `https://<你的用户名>.github.io/guide-mk-cns/pages/` 路径访问。
3. **自行部署**：将 `pages/` 目录部署到任何静态文件服务器。
